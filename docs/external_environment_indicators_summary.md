## 五、外部环境对中国债券的传导机制

### 1. 美债收益率 → 中国债券

**传导路径**：
- 美债收益率上升 → 中美利差缩小 → 跨境资本外流 → 人民币贬值压力 → 中国债券收益率上升

**时间滞后**：
- 通常滞后 1-3 个交易日

### 2. 美元升值 → 中国债券

**传导路径**：
- 美元升值 → 新兴市场资本外流 → 人民币贬值 → 中国债券收益率上升

**时间滞后**：
- 通常滞后 1-5 个交易日

### 3. 全球风险事件 → 中国债券

**传导路径**：
- 全球风险事件（地缘冲突、经济衰退等） → VIX 上升 → 避险资本流入中国 → 中国债券收益率下降

**时间滞后**：
- 通常立即传导（同一交易日）

### 4. 大宗商品价格 → 中国债券

**传导路径**：
- 大宗商品价格上升 → 全球通胀预期上升 → 央行加息预期上升 → 中国债券收益率上升

**时间滞后**：
- 通常滞后 3-10 个交易日

## 六、外部环境综合评估

### 1. 外部风险指数

**外部风险指数** = (美债10Y收益率 / 2.0) × (美元指数 / 100) × (VIX指数 / 20) × (地缘政治风险指数 / 100)

**风险等级**：
- **风险指数 > 1.5** → 外部环境高风险，中国债券面临下行压力
- **风险指数 1.0-1.5** → 外部环境中等风险，中国债券风险可控
- **风险指数 < 1.0** → 外部环境低风险，中国债券面临上行机会

### 2. 跨境资本流动预测

**资本流向判断**：
- **中美利差扩大 + 人民币升值** → 资本流入中国，债券收益率下降
- **中美利差缩小 + 人民币贬值** → 资本外流中国，债券收益率上升
- **全球风险事件 + VIX上升** → 避险资本流入中国，债券收益率下降

## 七、与其他维度的关联

### 与基本面(F)的关系
- **全球通胀上升** → 中国通胀预期上升 → 央行加息预期上升 → 中国债券收益率上升
- **全球经济衰退** → 中国经济增长预期下降 → 央行降息预期上升 → 中国债券收益率下降

### 与流动性(L)的关系
- **美联储加息** → 全球流动性收紧 → 中国央行可能跟随加息 → 中国债券收益率上升
- **美联储降息** → 全球流动性宽松 → 中国央行可能跟随降息 → 中国债券收益率下降

### 与债券供需(A)的关系
- **外资大量流入** → 现券需求增加 → 债券收益率下降
- **外资大量流出** → 现券需求减少 → 债券收益率上升

### 与市场情绪(M)的关系
- **全球风险事件** → 市场恐慌 → 风险偏好下降 → 债券需求增加
- **全球经济向好** → 市场乐观 → 风险偏好上升 → 债券需求减少
